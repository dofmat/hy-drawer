Arguments:

    Type
    : `Boolean`

Arguments:
1.  `animated` (optional)

    Type
    : `Boolean`

    Default
    : `true`

Arguments:
1.  `animated` (optional)

    Type
    : `Boolean`

    Default
    : `true`
